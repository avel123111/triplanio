★ Новая дизайн-система **Horizon** (2026-06-06) — замена старой "Atlantic" (Pavel считал её «дешёвой/непривлекательной»). Пока это СТАНДАРТ/прототип, в код приложения ещё НЕ перенесён.

Файл: `triplanio_new/DESIGN_SYSTEM_HORIZON_2026-06-06.html` — самодостаточный HTML UI-kit (токены, типографика, компоненты, переключатель light/dark, copy-on-click свотчей). Цель — единый стандарт, все экраны делать только на нём.

Ключевые решения:
- **Шрифты**: Unbounded (display/заголовки/вордмарк) + Onest (UI/текст). Оба с полной кириллицей. Заменяют старые Inter/Inter Tight.
- **Primary**: мягкий азур `#2675CF` (light) / `#6BA6FF` (dark) — светлее и «голубее» прежнего `#2167e2`. AA white-text 4.64.
- **Типы событий**: hotel=индиго `#5C63E0`, transfer=тил `#0E9FB0`, activity=коралл `#E27636`, car=изумруд `#149E6C`. У каждого `-ink` (затемнённый текст бейджей, AA на тинте) + soft/soft-2.
- **Контекст**: info/success/warning/danger, у каждого `-soft` тинт + `-ink` для AA-текста.
- **AI-паттерн**: холодный градиент azure→violet→magenta, solid `--ai` фиолетовый.
- **Pro-паттерн**: тёплое золото gold→amber (намеренный контраст с холодным AI) — бейджи/плашки/кнопки апгрейда.
- **Фоны**: light bg `#F4F7FB` / surface white; dark bg `#0B1018` / surface `#131A26`. Прохладные нейтрали с синим подтоном.
- Радиусы (sm8/btn11/card18/pill), тени soft/card/pop, моушн ease-out `cubic-bezier(.23,1,.32,1)`.

Весь контраст текст-на-фоне проверен скриптом ≥AA (≥4.5), обе темы отрисованы и выглядят выверенно. Связано с [[triplanio-ui-audit]], [[triplanio-design-ux-audit]], [[triplanio-editor-redesign-mockup]].

**v2 (Palette Lab)** — файл `triplanio_new/DESIGN_SYSTEM_HORIZON_V2_2026-06-06.html` (v1 НЕ удалён, лежит рядом). По просьбе Pavel «поиграть с цветом»:
- Шрифт **Manrope** (кириллица) вместо Unbounded+Onest.
- **7** переключаемых палитр (data-variant × data-theme), все AA white-text: **Azure** `#2675CF`, **Bloom**(rose) `#D33A72`, **Lagoon**(turquoise) `#0B7E8C`, **Iris**(violet) `#5A57E6`, **Ember**(coral/terracotta) `#C84A28`, **Forest**(emerald) `#0A8055`, **Graphite**(slate/editorial) `#455168`. (Forest разводит коллизию green: transfer→синий, car→тил. Graphite — низкочроматный premium-нейтрал, события остаются яркими.)

Таймлайн = леджер с цветными левыми кромками, редактор = таблица с колонками; переезды = пунктирные seam-линии; теги = прямоугольники с левой 3px-кромкой.

- 3 палитры primary (AA): **Cobalt** (мягкий синий, дефолт) `#3D6CD0`, **Clay** (тёплый кирпич) `#B5512E`, **Pine** (хвойный) `#1F7A6B` (разводит коллизию: переезд→синий).

**Lumo** — ОТДЕЛЬНАЯ дружелюбная дизайн-система (Meridian показался Pavel «грубоватым»: острые углы, серьёзный шрифт). Файл `triplanio_new/DESIGN_SYSTEM_LUMO_2026-06-06.html`.
- Язык: **friendly/тёплый/округлый** — крупные скругления (card 24px, btn 14px pill), пухлые мягкие тени, пружинистые hover (translateY+scale, ease-spring), дружелюбные иконки-плитки (squircle), большие тач-цели, мягкие градиентные блобы в hero, пилюли-переезды.
- Шрифты: **Rubik** (округлый display) + **Nunito** (text), обе с кириллицей. ⚠️Изначально был Fredoka, но Fredoka НЕ содержит кириллицу (заголовки падали в фолбэк) — заменён на Rubik, проверено document.fonts.check + рендером.
- **4** палитры primary (AA): **Sky** (мягкий синий, дефолт) `#2173C8`, **Pink** `#CE2F7B`, **Turquoise** `#0A7C8A`, **Grape** `#724ECB`. (Coral убран по просьбе Pavel, добавлены Pink+Turquoise.) Коллизии: Pink → активность становится фиолетовой; Turquoise → переезд становится синим.
- Activity = розовый `#E0568F` (не оранжевый). AI(violet)/Pro(gold) стабильны.
