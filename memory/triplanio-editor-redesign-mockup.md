Дизайн-направление для экрана «Редактирование структуры трипа» (TripStructureEdit) и его правых панелей (city / event view / edit / create). Pavel дважды просил именно ПОЛНЫЙ редизайн, а не перестановку текущих блоков.

**Утверждённый визуальный язык — «editorial concierge»:**
- Шрифты: Fraunces (сериф) для названий городов и хиро-заголовков; Inter Tight для UI-заголовков; Inter для текста. (В реальном репо есть Instrument Serif в warm-теме — сериф уместен.)
- Паттерны инспектора: фото-хиро с градиентом + сериф-тайтл; **stat-band** (значения через тонкие вертикальные разделители) ВМЕСТО боксовых KV-сеток; airy detail-rows на хайрлайнах; насыщенные строки с фото-миниатюрой (отель); тикетный boarding-pass для переезда; спокойная янтарная валидация (save не блокируется).
- Экран показан В layout приложения: хедер (бургер, крошки, автосейв, аватары участников, Поделиться, уведомления, юзер) + **скрываемое левое меню** (разделы трипа, сворачивается в icon-rail) + lens-бар (Маршрут/Карта/Календарь).
- **Drag-and-drop**: pointer-based сортировка городов (ручка-грип, lift+тень, расступание соседей, пересчёт номеров) и активностей.
- Канвас = структура (карточки городов: фото, сериф-имя, чипы отель/активности/предупр., степпер ночей, seam-коннекторы переездов). Инспектор = деталь выбранного объекта. Не дублировать.
- Карта живая: lens «Карта» + мини-карты в city/transfer; переезд подсвечивает/анимирует маршрут. (Связано с [[triplanio-editor-ui-redesign]] и live route preview.)
- Всё на токенах из src/design/app.css (brand #2167e2, --ev-hotel/transfer/activity/car), light-тема первична.
