## Зонтик над всем: план AARRR (TRIP-198)
Комплексный маркетинг-план по AARRR (скилл `marketing-plan`) = зонтик над всей маркетинг-таблицей. Файл-деливери = 15-листовый Excel, приложен в Linear **TRIP-198** (вне репо, правило #3): Обзор, Executive summary, Стратегическая рамка, Аудит текущего состояния (0-5), Acquisition/Activation/Retention/Referral/Revenue, 90-дневный роадмап, 12-мес горизонт, Ops-стек, Банк идей, Метрики+RACI, Сверка память.

**3 большие ставки:** (1) виральная петля шеринга (K>0.5, CAC~0); (2) вера вместо пруфа (демо-магия + reverse-trial + founder-письма) - binding constraint = вера, не цена; (3) два рынка/два рельса (EN через Stripe, RU через Telegram Payments).
**Волны:** 0 фундамент воронки -> 1 дешёвый трафик (SEO+comparison+лид-магниты) -> 2 всплеск (Product Hunt+каталоги). Трафик НЕ льём, пока воронка не конвертит.
**North Star:** активные поездки с >=2 участниками.
**Диагноз аудита:** сильны в стратегии (позиционирование/ICP/voice/pricing 3-4), слабы в исполнительном верхе воронки (SEO/контент/lifecycle/аналитика/пруф 0-1). 90 дней = достроить воронку и аналитику, не лить трафик в дырявое ведро.

## 5 открытых решений (нужны от Ильи/Pavel, лист 13 плана)
1. **CAC неизвестен** (highest-impact) - блокирует прогноз выручки.
2. **WTP RU не валидирован** - цена = гипотеза (5-8 интервью/сегмент + Telegram-поллы).
3. **Stripe без RUB -> Telegram Payments** - go/no-go + сроки (блокер монетизации RU).
4. **Lifetime / Founder deal** - go/no-go.
5. **Инструментация аналитики** - пишутся ли события воронки? Пререквизит ко всем метрикам.

Все 7 первичных исследований приложены к **Linear TRIP-198** как файлы (xlsx/ods/docx, вне репо). В памяти как топики - только часть:

- **Оффер / запуск (TRIP-184)** -> [[triplanio-launch-offer]] (в памяти, но **на rev2; вложение в TRIP-198 = rev4**: убраны групповой Pro-трип, «полгода в подарок», founder-lock - при использовании сверяйся с rev4-файлом в Linear).
