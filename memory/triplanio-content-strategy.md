★СТРАТЕГИЯ 2026-07-07 (скилл marketing-skills:content-strategy, TRIP-200). Контент-план поверх готовых [[triplanio-marketing-psychology]], [[triplanio-launch-offer]], [[triplanio-pro-model]] и AARRR-плана (TRIP-198). Заземлён на 7 маркетинг-файлов (позиционирование/УТП, VOC ~90 цитат, brand voice/5 pillars, ценообразование, оффер-меню, психология, AARRR) — сверены.

**Тема-зонт** = боль «всё разбросано»; несут 2 USP из ТЗ: AI-распознавание любого файла + алерты о дедлайне бесплатной отмены (последнее уникально, ложится на главный триггер «избегание потери»).

**5 контент-пилларов:** P1 «Собери поездку в одном месте» (анти-хаос) · P2 «ИИ соберёт брони» (распознавание, USP#1) · P3 «Не проспи дедлайн отмены» (алерты, USP#2, ГИПОТЕЗА) · P4 «Компанией без хаоса» (совместность+деньги, виральный) · P5 «Маршруты и вдохновение» (шаблоны/programmatic + «Моя статистика»). Один материал = 1-2 пиллара.

**3 канала-драйвера:** SEO+шаблоны маршрутов и Telegram RU/СНГ = co-#1 (компаунд-актив + primary-рынок/платёжный рельс); соцсети по сегментам (друзья/соло/пары) = усилитель виральной петли, сильнее после первого пруфа. Searchable-first (SEO фундамент), share усилитель. Демо-магия («письмо → таймлайн») — несущий формат, заменяет отсутствующий соц-пруф.

**Решения:** редкалендарь привязан к волнам AARRR (0 Фундамент / 1 Трафик / 2 Всплеск + непрерывно), не отдельная система. Языки = RU/EN-гибрид: темы RU-основные, EN только для comparison-страниц (TripIt/Wanderlog/Stippl) и EN-SEO (two-rails GTM). Скоринг тем 40/30/20/10.

**Флаги (не решено):** ценность Telegram-бота/напоминаний публично НЕ валидирована (VOC gap №1) → P3 и Telegram-USP помечены как гипотеза, валидировать поллами/интервью ДО крупной ставки; search-данные качественные (нет Ahrefs/GSC); Slack-коннектор из ТЗ не настроен. Красные линии соблюдены (не фабриковать пруф, без фейк-таймеров, дефис «-», строки через Tolgee).

**Деливери (правило #3 — не в репо):** приложены в Linear TRIP-200 — `TRIPLANIO_CONTENT_STRATEGY_TRIP-200.xlsx` (11 листов) + сводный Word-досье `TRIPLANIO_MARKETING_DOSSIER_TRIP-200` (все 8 маркетинг-файлов + стратегия в одном .docx, landscape, иерархия заголовков H1-H4). Файлы xlsx/docx собраны программно (в окружении нет офисных либ/python) — парсеры xlsx/ods/docx + zip-writer на node; формат делить с [[triplanio-design-deliverable-delivery]].
