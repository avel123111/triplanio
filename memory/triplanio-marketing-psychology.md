# Triplanio: психология маркетинга - справочный слой (TRIP-194)
_Скилл marketing-psychology. Не кампания, а reference-слой поверх бренда/оффера [[triplanio-launch-offer]] + [[triplanio-pro-model]]. Полный файл-деливери приложен в Linear TRIP-194 (вне репо, правило #3). Голос: «ты», не давим, не прячем free, только дефис «-» [[triplanio-feedback-no-em-dash]]._

## Суть
Binding constraint продукта = **вера** (MVP, ноль соц-пруфа), не цена. Значит триггеры несут **снятие риска + показ ценности ДО оплаты**, а не громкую ургентность.

## 4 несущих триггера (приоритет)
1. **Избегание потери (loss aversion) - ГЛАВНЫЙ.** «Не проспишь бесплатную отмену», «ничего не теряется в переписке», «видно, кто кому должен». Ядро ценности = предотвращённые потери. НЕ запугивание катастрофой.
2. **Дефицит / честная ургентность - только через РЕАЛЬНЫЙ дедлайн отмены брони / дату поездки / founder-lock.** Фейк-таймеры и «осталось N мест» ЗАПРЕЩЕНЫ.
3. **Принадлежность / единство - виральное топливо.** «Вся компания в одном месте по одной ссылке», «бесплатно для всех, кого пригласишь». Seat-guilt запрещён (приглашённые бесплатны намеренно).
4. **Соц-доказательство - сильный рычаг категории, но СЕЙЧАС недоступен (MVP).** Фабриковать нельзя. Замена: демо «магии» (письмо->AI-карточка) + прозрачность «мы только запустились» (pratfall). Строим честно по мере роста (реальные счётчики/отзывы) -> тогда переходит в несущий.

## Вторичные модели
zero-price (free=проба), present bias («AI за секунды» = проектируемый peak-end), anchoring (годовой $39.99 = единственный герой), mental accounting («$8.99/трип», «меньше чашки кофе»), goal-gradient/Zeigarnik (прогресс трипа 80%), reciprocity, endowment/IKEA (уже свой трип), activation energy («перешли письмо - остальное сами»), Hick/paradox of choice (один CTA), позитивный framing.

## Матрица поверхность->триггер
Лендинг hero=принадлежность+потеря; ценовая=anchoring; paywall бюджета/Telegram=потеря; приглашение/шеринг=принадлежность; онбординг=present bias+activation energy; письма/пуши=честный дедлайн+потеря; момент раннести=прозрачность/pratfall.

## Красные линии
Фейк соц-пруф; фейк-таймеры/«осталось N мест»; скрытый триал/внезапное списание; seat-guilt; инфобиз-давление/«$497 value»; dark-pattern дефолты; «—» в тексте.

## Зависимости
Платные триггеры «на полке» пока Stripe не сеттлит RUB (монетизация RU заблокирована); free-виральные катим раньше. WTP RU не валидирован -> про цену без «навсегда». Новые строки -> через Tolgee [[feedback-new-i18n-keys-go-to-tolgee]], не только JSON.
