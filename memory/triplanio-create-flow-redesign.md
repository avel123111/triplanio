**PR1 — СДЕЛАНО, подтверждено рабочим в редакторе (ждёт push dev+main).** Извлечён DnD-движок в общий хук `src/lib/useRouteDnD.js` (pointer-drag мышь-сразу/тач-long-press-430ms, FLIP, live-preview порядок, клавиатурный перенос). **Решение по reuse (важно):** ряд города в TSE на ~80% = брони/переезды/конфликты (`GridNode/SeamTransfer/hotelFor/actsFor/cityConflicts`), которых на создании НЕТ → НЕ делить ряд целиком.
