# Triplanio: оффер Pro/запуска (TRIP-184)
_Дизайн оффера через скилл marketing-offers, поверх утверждённой цены [[triplanio-pro-model]] (TRIP-181). Обосновано VOC, product marketing.docx, brand voice. Правило контента: только дефис «-», не «—»; обращение на «ты». Деливери — 2 xlsx в Linear (основной оффер rev2 + меню доп.офферов от 06.07), не в репо (правило #3)._

## Тип бизнеса и главный вывод
Triplanio = self-serve freemium SaaS → основную работу делает **цена/упаковка (TRIP-181, готово)**, конструкция оффера вторична. НЕ строить инфобиз-стайл (VOC тема K, brand voice).

**Binding constraint по Value Equation = Perceived Likelihood (вера, что сработает).** Dream outcome (спокойствие) сформулирован; Time delay низкий («вау» = AI заполнил карточку за секунды); Effort низкий (AI-ввод, приглашённые бесплатны). Слабое звено — **вера**: MVP, ноль соц-пруфа. → Лучшие офферы дают ощутить ценность ДО оплаты (проба хиро-фичи) и снимают тревогу «заброшу/сыро», а НЕ скидки/громкая ургентность.

## ★ИТОГ по MVP (после ревью Ильи 2026-07-06)
**Нового платного оффера для MVP сейчас НЕ добавляем.** Основной слой = существующая цена (TRIP-181) + уже реализованный в продукте down-sell на paywall (pro_trip $8.99 как меньший шаг для уходящих с подписки). Всё остальное — бэклог «на будущее» либо на обсуждение.

## Статус элементов оффера
**Исключено Ильёй (не делаем):**
- Денежные возвраты — 14 дней на подписку И на pro_trip (2026-07-03).
- Двусторонний реферал-бонус (2026-07-03). Виральность остаётся на бесплатном шеринге для приглашённых.
- Групповой Pro-трип «оплати раз — Pro для всех» (2026-07-06).
- Годовой = «полгода в подарок» framing (2026-07-06).

**Уже реализовано (в продукте):**
- Down-sell pro_trip на paywall — есть, отдельный оффер городить не нужно.

**На будущее (не MVP):**
- Проба распознавания до оплаты — узкий «первая бронь бесплатно» (1 распознавание на Free) ИЛИ широкий reverse-trial (вся первая поездка на Pro → откат на Free, без карты). Прямо в узкое звено (вера), работает даже при заблокированной RU-оплате (без карты).
- Контент-лид-магнит: шаблоны маршрутов + чек-лист «перед вылетом» (acquisition + SEO; не про оплату → мимо RU-блокера).
- Ранний доступ к мобилке для первых Pro; «основатель на связи»; win-back уснувших; сезонный/триггерный нудж.

**Открытые решения (ждём Илью):**
- **Lifetime / Founder deal** — разовый платёж → Pro навсегда (напр. $79). Кэш + «верящие» на MVP, НО бьёт по recurring-LTV и необратимо. Оставлено «на обсуждение».
- **Founder-lock цены** («цена ранних закреплена») — Илья убрал его из МЕНЮ доп.офферов (06.07), но он ещё присутствует в ОСНОВНОМ файле оффера rev2 (лист «Ургентность») как честная мягкая ургентность. Подтвердить: убирать ли и там, или оставить как элемент базового оффера. Молча основной файл не трогали.

## Что держит доверие без возвратов (risk-reversal)
Раз денежных возвратов нет — рычаг веры стоит на: **freemium как проба до оплаты** (free-tier = «триал», демо магии письмо→карточка) + **отмена в 1 клик без привязки** + **бессрочный pro_trip** (свойство продукта, не возврат). NB: без возврата рычаг слабее — следить за конверсией free→Pro.

## Бонусы = ответ на возражение VOC (задел на будущее)
«заброшу ещё одно приложение» → проба-до-вау; «не хочу вручную» → проба AI-распознавания; «друзья не поставят» → вход по ссылке без регистрации; «продукт сырой» → основатель на связи + ранний доступ к мобилке; «с чего начать» → шаблоны/чек-лист. Бонусы аддитивны, без инфляции «$497 value», макс 3-5.

## Блокеры и флаги
- **★БЛОКЕР оплаты (TRIP-181): Stripe не сеттлит RUB / не принимает РФ-карты**, а RU/СНГ — PRIMARY-рынок. Платные офферы (подписка, pro_trip, lifetime) для основного сегмента бесполезны без платёжного рельса. Бесплатные (проба без карты, лид-магнит) можно раньше. Отдельная GTM/платёжная задача.
- **WTP RU не валидирован** — цена = гипотеза; валидировать (5-8 интервью/сегмент + Telegram-поллы).
- **Ноль соц-пруфа (MVP)** — проба-до-оплаты + демо магии заменяют отзывы; не фабриковать пруф.
- Строки — только через Tolgee/`t()` (правило #4), не хардкод.
