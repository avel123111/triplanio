# Triplanio: оффер Pro/запуска (TRIP-184)
_Дизайн оффера через скилл marketing-offers, поверх утверждённой цены [[triplanio-pro-model]] (TRIP-181). Обосновано VOC-исследованием, product marketing.docx, brand voice. Утверждения Ильи ждём — это черновик-предложение, не финал. Правило контента: только дефис «-», не «—»; обращение на «ты»._

## Тип бизнеса и главный вывод
Triplanio = self-serve freemium SaaS → по скиллу marketing-offers основную работу делает **цена/упаковка (TRIP-181, готово)**, а конструкция оффера (бонусы/гарантии/ургентность) вторична и layer'ится поверх. НЕ строить «grand slam offer» в стиле инфобиза — travel-аудитория и brand voice это отторгают (VOC тема K, «не давим, не прячем free»).

**Binding constraint по Value Equation = Perceived Likelihood of Achievement (вера, что сработает).** Dream outcome (спокойствие, «всё в одном месте») сформулирован; Time delay низкий («вау» = AI заполнил карточку за секунды); Effort низкий (AI-ввод, приглашённые бесплатны, вход по ссылке). Слабое звено — **вера**: MVP, ноль соц-пруфа (product marketing §8.3), тревоги «ещё один планировщик — заброшу» / «безопасно ли отдавать документы». → Работа оффера = **снять риск и показать ценность ДО оплаты**, а не громкая ургентность.

## Оффер по 6 компонентам (анатомия)
1. **Core deliverable** = Pro как в TRIP-181 (AI-распознавание любого файла, Telegram-напоминания + AI-помощник, алерты отмены в Telegram, бюджет-сплит, безлимит трипов, in-app @assistant).
2. **Value-stack (обрамление, не инфляция):** формулировать Pro как «ИИ делает ручной ввод за тебя + напомним до дедлайна отмены + все в синхроне по одной ссылке», привязка к болям VOC. Без выдуманных «$497 value». Три SKU = три уровня commitment: годовой (герой) / месячный / pro_trip (импульс $8.99).
3. **Гарантия / risk-reversal (ГЛАВНЫЙ рычаг):**
   - **Возврат 14 дней без вопросов** на подписку Pro (честный freemium, анти-Wanderlog dark-pattern).
   - **pro_trip**: «не помог с этой поездкой — вернём» (14 дней). Плюс структурная гарантия — pro_trip бессрочный (не сгорает).
   - **Отмена в один клик, без lock-in** (Stripe cancel anytime). Явно проговорить (снимает тревогу «привяжусь»).
   - Freemium сам по себе = risk-reversal («пробуй бесплатно, сколько нужно»).
   - НЕ обещать outcome-гарантию («или доплатим») — скилл прямо запрещает over-promising.
4. **Мягкая ургентность (только честная):**
   - **Цена первых / founder-lock**: «цена запуска зафиксирована для ранних — не вырастет, когда поднимем». Честно, потому что WTP RU не валидирован → цена v1 = гипотеза, будет пересмотр (см. флаги TRIP-181).
   - Естественная сезонность: «спланируй поездку этим летом» — привязка к реальному трипу, не к таймеру.
   - **ЗАПРЕЩЕНО**: фейковые таймеры, «осталось 3 места», скрытый триал с внезапным списанием.
5. **Нейминг оффера** (brand voice, «ты», без жаргона): «Pro для первых» / «Тариф ранних путешественников»; реферал — «Позови друга — обоим Pro».
6. **Цена/структура** = TRIP-181 без изменений (годовой герой $39.99, месячный $6.99, pro_trip $8.99). Down-sell вход = pro_trip; reverse-trial только как A/B-эксперимент, не дефолт.

## Реферальный бонус обеим сторонам (виральная петля, GTM §7.1)
Механика двусторонняя: пригласивший и приглашённый оба получают Pro-бонус (напр. +1 месяц Pro каждому, или бесплатный pro_trip обоим). Усиливает уже существующее виральное топливо (шеринг бесплатен для приглашённых). Управляемо, без давления. ⚠️ Требует продуктовой доработки (учёт рефералов, выдача бонуса) — отдельная задача, не часть дизайна.

## Лончевое окно
Мягкое окно запуска (напр. первые N недель / до даты) с founder-lock ценой + реферальным бонусом, БЕЗ обратного отсчёта. Месседж: «Мы только запустились — заходи первым, цена ранних закреплена». Честно и в tone of voice.

## Блокеры и флаги (критично)
- **★БЛОКЕР оплаты (из TRIP-181): Stripe не сеттлит RUB / не принимает РФ-карты, а RU/СНГ — PRIMARY-рынок.** Любой платный оффер для основного сегмента бесполезен, пока нет платёжного рельса. Оффер-дизайн готов «на полке», но запуск монетизации для RU **заблокирован** на отдельной GTM/платёжной задаче. Реферальный/лончевый слой для не-платящих (free-виральность) можно катить раньше.
- **WTP RU не валидирован** — цена и founder-lock = гипотеза; валидировать (5-8 интервью/сегмент + Telegram-поллы) до «навсегда».
- **Ноль соц-пруфа (MVP)** — гарантия + демо «магии» (письмо→карточка) заменяют отзывы на старте; не фабриковать пруф.
- **In-app доставка алерта отмены во Free** может требовать доработки (инфра напоминаний Telegram-центрична).

## Что дальше (реализация — отдельные задачи, НЕ этот дизайн)
Публичная ценовая страница (копи под brand voice, годовой герой, pro_trip контекстно) — уже помечена как СЛЕД в TRIP-181; сюда же навесить гарантию-строку + founder-lock. Реферальная механика — продуктовая задача. Строки — только через Tolgee/`t()` (правило #4), не хардкод.
