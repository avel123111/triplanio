★МАКЕТ 2026-06-17 (в код НЕ перенесено). Единый полноэкранный HTML-прототип всех 10 экранов трипа БЕЗ шапок трипа (убраны и глобальный AppHeader, и per-screen TripScreenBar). Светлая тема Lumo (Rubik+Nunito, токены сверены с src/design/app.css). Навигация ТОЛЬКО через левое меню `app-side` (группы «Разделы»/«Управление» + Pro-карточка из реального TripSidebar). Адаптив: ≤680px сайдбар скрыт → нижняя нав (Обзор/Лента/Карта/Бюджет/Ещё) + bottom-sheet «Ещё»; на мобайле сверху мини-бар только с логотипом+названием экрана (это НЕ шапка трипа).

Решения по переносу контролов (Pavel делегировал «куда логично»):
- Бюджет: «Ручная трата» (primary+plus) и «Курсы» (ghost+arrowSwap), которые в коде идут в TripScreenBar через useTripScreenActions, перенесены в кластер вверху-справа summary-полосы; на мобайле «Ручная трата» = FAB, «Курсы» = клик по FX-стат-карточке.
- Планировщик (TripStructureEdit): «убрать карту» удалён (карта всегда сбоку); переключатель даты старта перенесён в шапку панели «Маршрут» как степпер «Старт − 8 июля +» (двигает даты всех городов).
- Документы: шапка+кнопка убраны, добавление = карточка-плейсхолдер/dropzone в сетке.
- Участники: шапка и дублирующая кнопка приглашения убраны, единственный invite — инлайн-строка внизу списка.
